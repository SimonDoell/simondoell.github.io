Real Time Liquid Particle Simulation using C++ with SFML

Description:

The amount of particles can be changed via a variable. Each particle gets added to a list

and a loop updates the accelaration depending on the proximity of other particles.

Performance is limited by single-core performance(i.e. only one cpu core gets used for all calculations, which is a commen limitation for C++ code).

Close particles get pushed away, while distant particles slightly attract each other.

This gives semi-realistic wave behavior, which also gets repelled by the walls.

SFML is only used for drawing and moving the shapes, not for any logic or calculations.

The higher the speed, the brighter the particles are.

Adaptive Real Time Gravity Simulation using C++ with SFML

Description:

This is a 2d gravity simulation using 2 lists for the objects emitting gravitationell force(colored blue) and ones attracted by it(colored white).

Limited by single-core performance. Can simulate about 15000 objects with smooth performance.

Best on the real inverse square algorithm for calculating the strenght of attraction.

When objects hit a object emitting a gravitationell pull, they get deleted from the list.
